Sync Tools

Run these commands in your terminal (VS Code → Terminal) to sync new products, generate AI descriptions, and update everything automatically.

1

Full Sync — Square API + AI Vision

Fetches all products from Square, generates AI descriptions from product images, updates your Square shop AND products.js

node sync-products.js --square=YOUR_TOKEN --key=YOUR_OPENAI_KEY
2

Download Images

Downloads all product photos from Square to your local images/ folder so they load fast on your website.

node download-images.js
3

Push to Website (GitHub Pages)

Commits everything and pushes live to sinuguru.github.io/Shop

git add . && git commit -m "Update products" && git push
★

One-liner — Do Everything At Once

node sync-products.js --square=YOUR_TOKEN --key=YOUR_OPENAI_KEY && node download-images.js && git add . && git commit -m "Update products" && git push
Only need to sync when you add new products on Square or want to refresh AI descriptions. Your website and Square shop update together automatically.

Fix Product URLs

Fetches your Square sitemap and automatically matches every product to its real individual product page URL. Run this whenever you add new products.

API Keys & Security

Admin Password

Password is stored as a SHA-256 hash in your browser — never as plain text. Choose something you won't forget; there is no recovery option.

OpenAI API Key

Used for AI captions, banner text, product descriptions & more. Get yours at platform.openai.com.

Runway ML API Key AI Creative Studio

Used to generate product images & Instagram/WhatsApp videos with AI. Get your key at dev.runwayml.com → Log in → Account → API Keys.

Square API Token

Used by the sync script to pull products from your Square shop. Get yours at developer.squareup.com → your app → Production Access Token.

Keys are stored in your browser only (localStorage) — never sent anywhere except directly to OpenAI/Square. Clear browser data to remove them.

Ready-to-run Sync Command

Your tokens pre-filled — just copy and run in terminal.

Save your keys above to generate this command.
